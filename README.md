📖 项目简介

本项目是一款借助AI辅助整合编写的多功能批量资源下载工具。整体界面是原生简约风格，上手简单，占用资源也比较小。目前基础下载相关功能都已经开发完成，逻辑运行稳定，很适合拿来学习或者自己二次改动。

本工具是n站与禁漫天堂综合资源下载器，整合双站下载能力，内置每周必看资源功能。
项目支持读取本地新建文本文档内的合集资源链接/ID进行批量解析下载，新建文本文档同时也可作为功能测试文件使用，方便调试程序各项下载逻辑。程序使用中遇到任何问题，欢迎随时提 Issues 反馈。

✅ 核心功能

• 可视化图形界面，新手也能快速上手操作

• 支持批量添加下载任务，单个任务失败可以手动重试

❓ 常见问题

1. 下载任务失败怎么办？

可以按照下面三点逐一排查：

1. 核对填写的代理地址和端口是否无误

2. 检查本地网络是否正常连通

3. 确认粘贴的资源ID没有多余空格或者错误字符

2. 怎么提高下载速度？

1. 选用线路稳定、带宽充足的代理节点

2. 保证自身本地网络环境良好，减少丢包延迟

3. 不要一次性开启太多并发任务，避免网络拥堵
