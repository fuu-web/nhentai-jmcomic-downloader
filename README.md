# nhentai.net 图片爬虫

一个功能强大的 nhentai.net 图片下载工具，支持图形界面和命令行两种使用方式。

## 主要特性

- **双模式支持**：GUI 图形界面 + 命令行
- **批量下载**：支持多个画廊连续下载
- **智能重试**：自动检测缺失页并重试下载
- **原图下载**：支持下载原始质量图片
- **悬浮胶囊**：便捷的悬浮下载器
- **剪贴板检测**：自动识别画廊ID
- **元数据保存**：自动保存画廊信息
- **错误汉化**：中文错误提示

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### GUI 版 (推荐)

```bash
python nhentai_gui.py
```

或双击 `nhentai下载器.exe`（如果存在）

**功能：**
- 图形界面操作，简单直观
- 批量下载多个画廊，连续下载
- 悬浮胶囊下载器（置顶可拖拽）
- 小火箭悬浮窗（磁吸屏幕边缘）
- 狂暴模式全站爬取
- 原图下载（非缩略图）
- 完整性检测 + 5轮重试（可调）
- 自动跳过已完成画廊
- 自动补齐未完成画廊
- 文件夹状态标记：`[完整]` / `[缺失X页]`
- 画廊元数据TXT文件
- 错误信息汉化（404/超时/代理等）
- Windows风格传输进度显示
- 日志区实时显示下载详情

### 命令行版

```bash
# 下载单个画廊
python nhentai_crawler.py 123456 -p http://127.0.0.1:7890

# 搜索下载
python nhentai_crawler.py "keyword" -s -p socks5://127.0.0.1:1080 -n 10
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `target` | 画廊ID或搜索关键词 |
| `-p, --proxy` | 代理地址 (支持HTTP/SOCKS5) |
| `-o, --output` | 输出目录 (默认: ./downloads) |
| `-q, --quality` | 图片质量: low/high (默认: low) |
| `-w, --workers` | 并发下载数 (默认: 3) |
| `-s, --search` | 启用搜索模式 |
| `-n, --number` | 搜索模式最大下载数 (默认: 5) |

### 代理格式

- HTTP代理: `http://127.0.0.1:7890`
- SOCKS5代理: `socks5://127.0.0.1:1080`
- 带认证: `http://user:pass@127.0.0.1:7890`

## GUI功能详解

### 主界面

- **配置区**：代理、输出目录、图片质量、重试轮数
- **输入区**：每行一个画廊ID或URL
- **按钮区**：开始下载、停止、狂暴模式、打开目录、胶囊开关
- **传输区**：显示当前下载进度、速度、剩余时间
- **文件列表**：左侧显示画廊目录内所有文件
- **日志区**：右侧实时显示所有操作详情

### 悬浮胶囊

- 点击主界面「胶囊」按钮显示/隐藏
- 置顶窗口，可拖拽到任意位置
- 双击展开面板，粘贴ID后回车自动下载
- 下载进度同步显示在主界面传输区和日志区
- 剪贴板检测：复制画廊ID后自动提示

### 狂暴模式

- 点击「狂暴模式」按钮打开配置
- 自动检测已下载的最大ID，从下一个开始
- 404单独计数，不计入失败
- 连续404或连续失败达上限自动停止
- 磁盘空间不足自动停止
- 所有下载详情实时显示在日志区

### 文件夹状态标记

下载完成后文件夹自动添加状态标记：
- `631125_Butter Bunny 合集 [完整]` — 全部下载成功
- `631125_Butter Bunny 合集 [缺失3页]` — 有缺失页

### 画廊信息文件

每个画廊目录内自动生成：
- `info.json` — 机器可读的元数据
- `画廊信息.txt` — 人类可读的信息（标题、艺术家、标签、语言等）
- `错误日志.txt` — 下载错误详情（仅在有错误时生成）
- `缺失页报告.txt` — 缺失页详情（仅在有缺失时生成）

## 错误信息汉化

| 错误 | 中文提示 |
|------|----------|
| 404 | 页面不存在，该画廊或图片已被删除 |
| 403 | 访问被拒绝，可能需要更换代理 |
| 429 | 请求过于频繁，请稍后再试 |
| ConnectionError | 网络连接失败，请检查网络或代理 |
| Timeout | 请求超时，请检查网络或更换代理 |
| ProxyError | 代理连接失败，请检查代理地址 |

## 注意事项

1. **代理设置**：由于网络限制，通常需要使用代理才能正常访问
2. **下载频率**：建议适当控制下载频率，避免被封IP
3. **存储空间**：高清图片占用较大空间，请确保有足够存储空间
4. **法律风险**：请遵守当地法律法规，仅用于个人学习研究

## 常见问题

### Q: 为什么下载失败？
A: 请检查：
- 代理设置是否正确
- 网络连接是否正常
- 画廊ID是否有效
- 是否被网站限制（尝试更换代理）

### Q: 如何提高下载速度？
A: 可以尝试：
- 使用更稳定的代理
- 增加并发下载数（注意不要太高）
- 选择低质量模式（缩略图）

### Q: 文件命名规则是什么？
A: 格式为：`{画廊ID}_{标题} [状态标记]`
- `[完整]`：所有页面下载成功
- `[缺失X页]`：有X页下载失败
