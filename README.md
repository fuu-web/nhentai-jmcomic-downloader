📖 项目简介

本项目是一款由 AI 辅助编译、整合开发的多功能资源批量下载工具。
软件采用纯原生简约图形界面，操作直观、轻量化、适配日常使用场景。
目前已实现绝大多数基础下载核心功能，功能完整、逻辑稳定，适合新手学习和二次开发。

✅ 核心功能

• 可视化图形界面操作，简单易上手

• 批量任务下载功能，支持手动重试任务

• 悬浮窗批量下载辅助功能

• 自动跳过已完成任务，避免重复下载

• 自动补全、接续未完成的中断任务

• 自定义文件夹管理、路径编辑功能

🌐 代理网络支持

本工具支持自定义代理网络配置，适配多种网络协议，解决特殊网络环境下载问题：

• 支持 HTTP 代理（含账号密码认证代理）

• 支持 SOCKS5 代理

• 使用方式：填写本地代理软件后台监听地址与端口即可生效

❓ 常见问题

1. 下载任务失败怎么办？

可依次排查以下问题：

1. 代理地址、端口配置是否准确

2. 本地网络连接是否稳定、通畅

3. 资源标识 ID 是否正确粘贴、无多余空格符号

2. 如何提升下载速度？

1. 更换并使用高稳定性、高速代理节点

2. 保证本地网络无延迟、无丢包

3. 减少同时并发任务数量，避免拥堵
